# Carat Money - Gold Calculator

A fully working gold calculator website based on Karatkash.com design.

## Features

✨ **Working Calculator**
- Real-time weight & purity calculations  
- Automatic 3% jewellery deduction
- Shows gross → deduction → final amount

🌍 **8 Languages**
- English, Bengali, Hindi, Kannada, Malayalam, Marathi, Tamil, Telugu

💰 **Rate Calculation**
- Base 24K rate: ₹13,500/gram
- Automatically calculates rates for other purities
- Formula: Weight × Purity% × ₹13,500

📱 **Responsive Design**
- Works on desktop, tablet, mobile
- Modern dark theme with gold accents
- Professional UI

---

## Formula

```
Gross Value = Weight (g) × Purity % × ₹13,500
Jewellery Deduction = Gross Value × 3%
You Receive = Gross Value - Deduction
```

### Example: 10g, 22K Gold
- Purity: 22K = 91.6%
- Gross = 10 × 0.916 × 13,500 = ₹123,660
- Deduction (3%) = ₹3,710
- Final = ₹119,950
